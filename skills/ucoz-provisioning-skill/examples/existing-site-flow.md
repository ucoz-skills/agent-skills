# Example: existing site scenario

User: Connect my existing uCoz site to MCP.

Agent:
1. Requests site URL.
2. Requests uCoz email/password.
3. Opens the site control panel.
4. Reviews available/active modules.
5. Clarifies which modules to enable before uAPI/FTP.
6. Creates or reuses a uAPI key.
7. Configures or reuses FTP credentials.
8. Assembles MCP config.
9. Verifies readiness.
10. Stops or hands off to the next skill.

Raw secrets are not printed in chat.
