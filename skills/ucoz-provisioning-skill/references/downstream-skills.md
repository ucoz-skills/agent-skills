# Downstream Skills

This helper finishes at MCP-ready site provisioning. It may hand off to a downstream skill only after the user explicitly asks for a follow-up task.

Default public package registry is empty. Examples of possible downstream task categories:

- landing/content generation;
- news/blog automation;
- users and permissions;
- forms and notifications;
- ecommerce/catalog tasks.

Do not hard-code a downstream skill in this public helper package. Add one only in a private/local registry for a concrete environment.
